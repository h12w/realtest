RealTest: real test environment for Go
======================================

h12.me/realtest provides real test environment for Go unit testing, based on Docker.

Includes:

* Queue
  - Kafka cluster (with zookeeper)
* Database:
  - MySQL
  - MongoDB
  - InfluxDB
  - ElasticSearch
* Cache
  - Redis
* Configuration service
  - ZooKeeper
